My Projects

Choir Practice App · Full-Stack Web Application · Bachelor's Thesis

A web application built to support choir rehearsal, from individual practice to conductor-led session management. The app renders interactive MusicXML scores in the browser, handles audio playback and recording, and provides separate interfaces for conductors and ensemble members with role-based access control. I designed and built the entire system independently, ran a six-participant usability study, iterated on the design based on structured feedback, and documented the process end to end to academic standard.

Tech: React, TypeScript, Node.js, MusicXML rendering, audio Web APIs

Empathic AI Chatbot · Research Project · FHNW x Horizon Europe (IMPortant)

A conversational AI system designed to support cancer patients, developed as part of a EU-funded research project. I was responsible for the full AI layer: a retrieval-augmented generation pipeline using vector embeddings for context retrieval, structured prompt engineering to shape empathic responses, and an evaluation framework based on EPITOME to measure empathy quality and identify weak outputs systematically. The goal was not just to build a chatbot but to build one that could be measured and improved.

Tech: Python, OpenAI API, vector databases, prompt engineering, EPITOME evaluation framework

Artifact of Earth · 3D Game · Unity

A physics-based 3D game built in Unity, developed as a personal long-term project. Artifact of Earth explores environmental themes through mechanics that reward curiosity and exploration.
